# ⏳ AgeWise — Age Calculator

A fast, precise, and beautifully animated age calculator built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** [AgeWise on GitHub Pages](https://mianhassam96.github.io/AgeWise/)
👨‍💻 **Author:** [MultiMian](https://github.com/Mianhassam96/)

---

## ✨ Features

- **Single Age Check** — Enter a name and date of birth to get a full age breakdown from years down to seconds, updating live every second
- **Age Comparison** — Compare two people's ages, see who is older, and get the exact age difference
- **Full Precision** — Years · Months · Days · Hours · Minutes · Seconds
- **Total Stats** — Total months, weeks, days, hours, minutes, and seconds lived
- **Born Day** — Shows the day of the week the person was born
- **Birthday Countdown** — Live countdown to the next birthday
- **Contact Form** — Built-in contact section with validation
- **Animated UI** — Floating orbs, shimmer effects, word-entrance hero, gradient borders
- **Mobile Responsive** — Works cleanly on all screen sizes

---

## 🚀 Getting Started

Just open `index.html` in your browser — no build tools or dependencies required.

```bash
git clone https://github.com/Mianhassam96/AgeWise.git
cd AgeWise
# open index.html in your browser
```

Or serve locally:

```bash
npx serve .
```

Then visit `http://localhost:3000`
